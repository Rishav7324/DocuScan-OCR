# APK delivery

Drop the release APK here as:

    DocuScan-OCR-v2.0.0.apk

The website's "Download" button links directly to `apk/DocuScan-OCR-v2.0.0.apk`,
so once the file is present users get a 1-click download.

Build it from the Android project (`./gradlew assembleRelease`) or download the
signed APK from your release pipeline, then place it in this folder and redeploy.
