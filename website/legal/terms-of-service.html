5. Description of the App

DocuScan OCR is a document scanning application for Android that provides the following core functionality:

  • Camera-based document capture — using the device's camera to photograph documents.
  • On-device optical character recognition (OCR) — using Google ML Kit to extract text from captured images. No network request is made for OCR processing.
  • Perspective correction and image filtering — tools to adjust and enhance scanned document images.
  • Organizational features — folders, document names, page management.
  • Private folders with encryption — PIN-protected folders with AES-256-GCM at-rest encryption.
  • Export to PDF, DOCX, and TXT — industry-standard formats for document sharing.
  • Optional cloud sync — manual or automatic backup to Cloudflare R2, Google Drive, or Dropbox. You configure and control this feature.

The App is provided free of charge and does not include any paid features, subscriptions, in-app purchases, or advertisements.
